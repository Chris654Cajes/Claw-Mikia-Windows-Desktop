🚀 FIRST RUN
═══════════════════════════════════════════════════════════════════════════════

When you launch the app:

1. A new database (MusicVault.db) will be created automatically
2. The interface appears with empty library
3. Click "📁 Add Folder" button
4. Select folder with your music files
5. Wait for scan to complete
6. Your library populates with songs
7. Search, browse, play, and enjoy!

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

"dotnet: command not found"
  → Install .NET 6.0 SDK from https://dotnet.microsoft.com/download

Build fails with errors
  → Run: dotnet clean && dotnet restore && dotnet build

No songs appearing in library
  → Click "📁 Add Folder" and select folder with music
  → Verify files are MP3, WAV, FLAC, AAC, OGG, or M4A

Database errors
  → Delete MusicVault.db (app recreates on next run)
  → Your music files are never affected

See BUILD_GUIDE.md for detailed troubleshooting
